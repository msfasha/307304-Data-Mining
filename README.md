# 307304 Data Mining Course

Welcome to the repository for the **307304 Data Mining Course**! This repo contains all the code, datasets, and materials used during the course. Below is an outline of the content organized by modules and topics covered.

## Repository Structure

### Datasets
- `apartment_prices.csv`: Dataset containing apartment price data for regression analysis.
- `BostonHousing.csv`: Classic dataset used for various regression and data mining tasks.

### Part 1: Introduction
- `1_Introduction to Data Mining.pdf`: Course introduction and overview.
- `1_Introduction to Data Mining.pptx`: Slides covering the fundamentals of data mining.

### Part 2: Regression
- `2-1-simple linear regression.ipynb`: Notebook covering simple linear regression techniques.
- `2-2-multiple linear regression.ipynb`: Notebook covering multiple linear regression analysis.
- `2-3-converting categorical to numerical.ipynb`: Converting categorical features into numerical ones for machine learning.
- `2-4-bias_variance_tradeoff.ipynb`: Exploring the bias-variance tradeoff in regression models.
- `2-4-feature_selection.ipynb`: Techniques for selecting the most relevant features for your regression models.
- `2-4-non-linear models.ipynb`: Application of non-linear regression models.
- `9-generate-sample-dataset.ipynb`: Generating sample datasets for practice.

### Part 3: Classification
- `3-1-classification_1.ipynb`: Introduction to classification algorithms and models.
- `3-2-classification_2.ipynb`: Continuation of classification techniques.
- `decision_tree.ipynb`: Implementation of decision tree classifiers.
- `knn.ipynb`: k-Nearest Neighbors algorithm for classification tasks.
- `logistic_regression.ipynb`: Logistic regression for binary classification problems.

### Part 4: Clustering
- `4-1-kmeans algorithm.ipynb`: K-means clustering algorithm for unsupervised learning.
- `4-2-dbscan algorithm.ipynb`: DBSCAN clustering algorithm for density-based clustering.

### Part 5: Association Rules Mining
- `5-1-association_rules_mining.ipynb`: Mining association rules from transactional datasets.

### Part 6: Dimensions Reduction
- `6-1-principal_components_analysis.ipynb`: Principal Components Analysis (PCA) for dimensionality reduction.
- `6-2-tsne.ipynb`: t-SNE for visualizing high-dimensional datasets.

### Part 7: Time Series Analysis
- `time_series_analysis.ipynb`: Notebook on time series data analysis techniques.

### Part 8: Text Mining
- `text_mining.ipynb`: Introduction to text mining and natural language processing.

### Part 9: Big Data Mining
- `bigdata_analytics.ipynb`: Techniques and tools for mining large datasets.
